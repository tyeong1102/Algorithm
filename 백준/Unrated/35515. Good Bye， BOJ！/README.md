### 문제 설명

<p>다음 중 하나 이상의 문자열을 부분 문자열로 포함하는 문자열 $S$를 구하여라. 단 알파벳의 경우, 대소문자를 구별하지 않는다. </p>

<ul>
<li>BOJ</li>
<li>Baekjoon</li>
<li>백준</li>
</ul>

<p>문자열 $A$가 문자열 $B$의 부분 문자열이라는 것은, $B$의 앞과 뒤에서 문자를 적절히 삭제했을 때 $A$가 되도록 할 수 있다는 뜻이다. 즉, $A$가 $B$의 연속한 구간으로 나타난다면 $A$는 $B$의 부분 문자열이다.</p>

### 입력 

 <p>입력은 없다.</p>

### 출력 

 <p>$S$를 출력한다. 만약 답으로 가능한 $S$가 여럿이라면, 아무거나 하나 출력한다. </p>
